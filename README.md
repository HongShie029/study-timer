# study-timer

공부 타이머

안드로이드 앱 실습 기말 구상
기능
필수
타이머 작동, 중지, 삭제
집중시간 기록(외부 시간 불러오기 필수)
유저가 시간 조정
--필요 지식
타이머 구현
외부시간 불러오기
유저와 상호작용해서 변수 조정

계획(4주, 28일)
1주차
할일: 계획및 아이디어 정리(같이 개발할 방법정리도)
예은: 계획및 구상
소영: 아이디어및 기능 생각

2주차
할일: ui디자인 완성및 필수 기능 개발 (이떄 제출 가능 상태 만들어야 함)
역할1: ui구상/구현, 집중시간 기록(외부시간 불러오기)
역할2: 타이머 제작, 데이터 저장(조정필요)

3주차
항일: 집중 휴식 구분 및 정지설정, 원형시간바, 4주차 ui구상(4주차도 어느정도 진행해야함)
역할1: 원형시간바, 4주차 ui구상
역할2: 집중 휴식 구분및 정지 설정

4주차
할일: 화면꺼진상태와 정지상태 구분, 그 외 선택 사항 구현(시간 남으면 기타 선택사항 전부구현 및 버그 수정)
역할분배 없음 그때봐서 유동적으로
